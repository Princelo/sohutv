# 搜狐视频网页版封装

## 原理

伪装浏览器的User Agent成Android设备（目前代码中是Nexus One Android 2.3.4版的UA），再向服务器发起请求时，服务器会切换到HTML5视频模式，视频就可以正常播放了。
